# C. 认证与账户模块（新增 4 页）

### 8) 认证模块（4 页）

* `/login` 登录页 (AuthLayout)
* `/register` 注册页（可选） (AuthLayout)
* `/profile` 个人中心 (T3/简化)
* `/auth/forgot-password` 忘记密码（预留） (AuthLayout)

---

## `/login` 登录页 (AuthLayout)

**用途**：系统统一登录入口。

**页面结构**

* 居中登录卡片
* Logo + 系统名称
* 输入框：邮箱/用户名
* 输入框：密码
* 复选框：记住登录
* 登录按钮
* 错误提示区
* 页脚版权信息

---

## `/register` 注册页 (AuthLayout)

**用途**：SaaS 自助注册（可选）。

**页面结构**

* 公司名称
* 管理员邮箱
* 密码
* 确认密码
* 注册按钮
* 注册成功提示

---

## `/profile` 个人中心 (T3/简化)

**用途**：个人信息与安全设置。

**页面结构**

* 顶部：用户名 + 状态标签
* 主区左：基本信息（邮箱、角色）
* 主区中：修改密码表单
* 下方 Tab：

  * 登录记录
  * 安全设置
* 保存按钮

---

---

# D. 租户与权限管理模块（新增 3 页）

### 9) 租户与用户管理（3 页）

* `/settings/tenant` 租户设置页 (T3/变体)
* `/settings/users` 用户管理工作台 (T2)
* `/settings/roles` 角色权限配置（简化 RBAC） (T2/变体)

---

## `/settings/tenant` 租户设置页 (T3/变体)

**用途**：租户级信息与报价模板设置。

**页面结构**

* 顶部：租户名称 + 保存按钮
* 主区左：公司基本信息（名称、地址、联系方式）
* 主区中：Logo 上传
* 下方 Tab：

  * 报价模板设置
  * 默认税率
  * 默认页脚
* 保存按钮

---

## `/settings/users` 用户管理工作台 (T2)

**用途**：租户内账号管理。

**页面结构**

* 顶部：标题 + 新增用户按钮
* 工具条：搜索 + 角色筛选
* 主区：表格

  * 用户名
  * 邮箱
  * 角色
  * 状态
  * 操作（编辑/禁用）
* 右侧抽屉：编辑用户

  * 修改角色
  * 重置密码
  * 启用/禁用

---

## `/settings/roles` 角色权限配置 (T2/变体)

**用途**：轻量 RBAC（Role-Based Access Control，基于角色的访问控制）。

**页面结构**

* 左侧：角色列表（Admin / Operator / Viewer）
* 右侧：权限勾选区

  * SKU 权限
  * 库存权限
  * 采购权限
  * 销售权限
  * 报价权限
  * API 权限
* 保存按钮

---

---

# E. 开放平台模块（新增 3 页）

### 10) API 与开发者中心（3 页）

* `/settings/api-clients` API 客户端管理 (T2)
* `/settings/api-logs` API 调用日志 (T2)
* `/developer` 开发者中心 (T3/展示型)

---

## `/settings/api-clients` API 客户端管理 (T2)

**用途**：管理 OAuth2 客户端。

**页面结构**

* 顶部：创建客户端按钮
* 表格：

  * client_id
  * 描述
  * scopes
  * 状态
  * 创建时间
  * 操作（重置密钥/禁用）
* 右侧抽屉：

  * client_secret（一次性展示）
  * Scope 勾选
  * 启用状态

---

## `/settings/api-logs` API 调用日志 (T2)

**用途**：API 审计与调用监控。

**页面结构**

* 筛选区：

  * 时间范围
  * client_id
  * 状态码
* 表格：

  * 时间
  * client_id
  * Query 类型
  * 查询复杂度
  * 状态
  * 错误信息

---

## `/developer` 开发者中心 (T3/展示型)

**用途**：API 使用说明与示例。

**页面结构**

* 顶部：标题
* 主区：

  * Scope 说明
  * GraphQL 示例查询
  * 速率限制说明
  * 错误码说明
* 可嵌入 GraphQL Playground（权限控制）

---
