# miniERP 页面级原型规格

## A. 通用页面模板（4 种布局）

### T1) 概览页模板 OverviewLayout
**用途**：只负责“指路”，不负责“干活”。
**页面结构**
* 左侧：主导航（模块列表）
* 顶部：页面标题 + 搜索框（按该模块搜索）+ 主按钮（新建/过账）
* 主区 A：KPI 卡片区（3~4 张）
* 主区 B：待办/异常区（可点击跳转到工作台并带筛选）
* 右侧：快捷操作卡（“新建X / 最近模板 / 常用过滤”）
* 底部：最近动作时间线（可跳详情）

### T2) 列表工作台模板 WorkbenchLayout
**用途**：主战场，高密度信息展示与批量处理。
**页面结构**
* 左侧：主导航
* 顶部：标题 + 主按钮 + 次按钮组（导入/导出/批量）
* 工具条：全局搜索 + 常用过滤（下拉/多选）+ 高级过滤折叠
* 主区：表格/列表（高密度，支持列显隐、排序、分页/虚拟滚动）
* 右侧：详情抽屉（选中行即显示，含关键字段 + 快捷操作）
* 底部：批量操作浮条（选中多行出现）

### T3) 详情页模板 DetailLayout
**用途**：单据/实体详情展示及关联信息整合。
**页面结构**
* 顶部：标题（编号/名称）+ 状态标签 + 关键动作按钮（编辑/作废/过账）
* 主区：信息分栏（左：基本信息；中：明细；右：关联/风险提示）
* 下方：Tab（明细/流水/附件/操作日志）
* 右侧（可选）：快捷动作（复制、生成关联单据）

### T4) 过账工作流模板 WizardLayout
**用途**：入库/出库/盘点等分步重操作。
**页面结构**
* 顶部：步骤条 Stepper（1 基础信息 → 2 明细 → 3 校验/差异 → 4 过账）
* 主区：表单/明细编辑（支持扫码/搜索添加）
* 右侧：汇总预览（行数、总数量、影响库存、异常提示）
* 底部：固定操作条（保存草稿 / 过账 / 作废）

## B. 页面清单（共 30 页）

### 1) 工作台首页（1 页）
* `/` 工作台首页 (T1)

### 2) SKU 与映射（6 页）
* `/skus/overview` SKU 概览 (T1)
* `/skus` SKU 列表工作台（默认入口） (T2)
* `/skus/new` 新建 SKU 向导 (T4/简化版)
* `/skus/:id` SKU 详情 (T3)
* `/skus/:id/edit` 编辑 SKU (T3)
* `/mappings` 外部料号映射工作台 (T2)

### 3) 采购（6 页）
* `/purchasing/overview` 采购概览 (T1)
* `/purchasing/po` PO 列表工作台 (T2)
* `/purchasing/po/new` 新建 PO (T4)
* `/purchasing/po/:id` PO 详情 (T3)
* `/purchasing/grn` GRN 列表工作台 (T2)
* `/purchasing/grn/new` 新建入库 GRN（过账工作流） (T4)
* `/purchasing/grn/:id` GRN 详情 (T3)

### 4) 销售/出库（6 页）
* `/sales/overview` 销售/出库概览 (T1)
* `/sales/so` SO 列表工作台 (T2)
* `/sales/so/new` 新建 SO (T4)
* `/sales/so/:id` SO 详情 (T3)
* `/sales/out` 出库单列表工作台 (T2)
* `/sales/out/new` 新建出库 OUT（过账工作流） (T4)
* `/sales/out/:id` OUT 详情 (T3)

### 5) 库存（3 页）
* `/inventory` 库存查询工作台 (T2)
* `/inventory/ledger` 库存流水审计工作台 (T2)
* `/inventory/reorder` 补货建议工作台 (T2)

### 6) 盘点（3 页）
* `/stocktake` 盘点单列表工作台 (T2)
* `/stocktake/new` 新建盘点（过账工作流） (T4)
* `/stocktake/:id` 盘点详情 (T3)

### 7) 设置（3 页）
* `/settings` 设置入口页 (T1/简化)
* `/settings/master-data` 主数据配置 (T2/变体)
* `/settings/system` 系统配置 (T2/变体)
